# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅         |

## Reporting a Vulnerability

如果你发现了安全漏洞，**请不要通过公开 Issue 提交**。

请通过以下方式私密报告：

- 发送邮件至：`agent@tuniu.com`
- 或使用 GitHub 的 [Private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability) 功能

我们会在收到报告后尽快确认并响应（通常 48 小时内）。

## Scope

以下属于安全关注范围：
- API Key 泄露风险（配置文件、日志输出中的敏感信息）
- 依赖链中的已知漏洞
- 命令注入或不安全的输入处理
